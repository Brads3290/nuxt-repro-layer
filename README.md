# Getting started

## Install

```bash
npm install --save-dev nuxt-lego

# Using yarn
yarn add --dev nuxt-lego
```

Requires Nuxt >= 3.1.0.

## Register Layer

_nuxt.config.ts_

```ts
export default defineNuxtConfig({
  extends: ['nuxt-lego'],
})
```
